код React-a лежит в "resources/js"


## Installation


```
$ git clone https://github.com/Khalif-abd/laravel-react-redux
$ composer install
$ php artisan migrate
$ php artisan passport:install
$ php artisan db:seed
$ php artisan db:seed --class=CurrencyTableSeeder
$ npm install
$ php artisan serve 
$ npm run dev 

```
